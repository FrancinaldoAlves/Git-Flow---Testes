# Git-Flow---Testes

Comandos do GitHUb e e GitFlow:

git clone: clona um repositório na máquina do usuário. 
<br>
git add . : adiciona os arquivos em uma área de "staging".
<br>
git commit: salva as informações de forma permanente no repositório local.
<br>
git status: serve para ver o estado atual do repositório.
<br>
git branch: lista todas as branchs locais.
<br>
git branch -m: renomeia a branch atual.
<br>
git push: sobe os arquivos salvos localmente para o repositório virtual.
<br>
git switch: serve para trocar de branch
<br>
git checkout -b: cria uma nova branch e troca pra ela.
<br>
git checkout --: desfazer alterações de um arquivo.
<br>
git checkout nome-commit: voltar para um commit.
<br>
git flow feature start: cria uma nova branch baseada na develop.
<br>
git flow feature finish: exclui a branch feature e faz merge para a develop.
